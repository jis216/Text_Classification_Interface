## Requirements
Python 2.7
flask
tarfile
sklearn
matplotlib
numpy
seaborn
nltk
download nltk 'english' stopwords

## How to run
in the root directory of this folder, run the following command:
python servery.py

Then use your brower to go to the local portal displayed on the terminal.
(default: http://127.0.0.1:5000/)
